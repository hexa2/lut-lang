lut-lang
========

Lutin Language Interpreter

Installation
------------

```
git clone git@github.com:hexa2/lut-lang.git
cd lut-lang
make
make test
```

Usage
-----

```
./bin/lut <LUT-FILE> \
  -o # Optimize/Transform code \
  -p # Print transformed code \
  -a # Static Analysis \
  -e # Execute/interpret code
```

That's all !
